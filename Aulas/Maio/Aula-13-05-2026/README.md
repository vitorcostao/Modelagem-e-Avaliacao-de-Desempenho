# Enunciado do Trabalho: Planejamento de Capacidade

## Tema
**Planejamento de Capacidade - Definindo SLAs para Infraestrutura nas Nuvens.**

## Contexto
Considere que é necessário negociar com o provedor de nuvem os melhores **SLAs** (*Tempo de resposta*, *Taxa de processamento* e *Disponibilidade de serviços*) para atender aos requisitos de desempenho, orçamento disponível e de qualidade (Utilidade) do serviço para uma empresa.

## Requisitos do Trabalho
Para a realização desta atividade, o grupo deverá:

1.  **Propor e Contextualizar uma Empresa Fictícia:** Definir uma organização que opte pela utilização de serviços em cloud.
2.  **Determinar os SLAs Adequados:** Estabelecer metas para:
    * Tempo de resposta;
    * Taxa de processamento;
    * Disponibilidade.
    * *Nota: Deve-se justificar estas escolhas com base no tipo de serviço oferecido aos usuários.*
3.  **Estimar Valores de Mercado:** Pesquisar ou estimar custos aplicados por provedores de cloud por cada requisição.
4.  **Restrição Financeira:** O orçamento disponível é de **R$ 20.000,00 por mês**.
5.  **Tomada de Decisão (Priorização):**
    * O grupo deve escolher o que priorizar: tempo de resposta, taxa de processamento, disponibilidade ou uma combinação destes.
    * A decisão deve ser devidamente justificada.

## Objetivo da Simulação
Realizar simulações para encontrar os melhores SLAs que favoreçam a melhor **qualidade (Utilidade)** do serviço a ser contratado com o melhor custo possível. 
*Atenção:* Note que, para atender aos requisitos, pode ser necessário pagar mais por cada requisição esperada.

## Entrega
Preparar um relatório de **no máximo 3 folhas** que analise os achados e as decisões tomadas.
